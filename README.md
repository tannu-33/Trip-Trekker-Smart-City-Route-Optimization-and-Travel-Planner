# Trip-Trekker-Smart-City-Route-Optimization-and-Travel-Planner
Developed TripTrekker, a C++ graph-based city tour planner implementing Dijkstra, DFS path search, and Branch-and-Bound TSP to compute optimal travel routes under dynamic traffic conditions.  Integrated Greedy Activity Scheduling, custom QuickSort to generate conflict-free daily itineraries, for realistic route optimization.
